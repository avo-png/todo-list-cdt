This is a Todo List app created for the Code the Dream React branch assignment. This app uses the latest Vite Using CLI as of July 13 2026.

## Description

This project contains the exercises for the Code the Dream React curriculum.

## Installation
1. Clone the repository:

   ```bash
   git clone https://github.com/avo-png/react-curriculum-v4-exercises.git
   ```

2. Navigate to the project directory:

   ```bash
   cd react-curriculum-v4-exercises
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

## Running the Development Server

Start the development server:

```bash
npm run dev
```

Then open the URL displayed in the terminal (usually `http://localhost:5173`) in your browser.
